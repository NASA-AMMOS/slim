# About GenAI Marketplace

> _Enabling GenAI development at JPL through AI-powered best practices and automation._

## What is GenAI Marketplace?

**GenAI Marketplace** is a Jet Propulsion Laboratory initiative that transforms how teams adopt GenAI development best practices. Instead of lengthy documentation, GenAI Marketplace delivers best practices as **automated solutions** that install and configure themselves in seconds.

## How It Works

### For Users

1. **Browse** the [marketplace](https://github.jpl.nasa.gov/pages/GenAI/genai-marketplace/)
2. **Install** via AI tool (e.g., for Claude Code: `/plugin install skill-name@genai-marketplace`)
3. **Use** by asking your AI assistant

**Compatible AI Tools:**
- Claude Code
- Cursor
- Windsurf
- Aider
- Other AI assistants with skill/MCP support

### For Contributors

GenAI Marketplace is community-driven. [Contribute](/docs/contribute/submit-ai-plugin) by:
- Creating new skills or agents
- Improving existing best practices
- Sharing your expertise
- Providing feedback on priorities
