# Contributing to GenAI Marketplace

Thanks for your interest in contributing! GenAI Marketplace is a community-driven project and we welcome all contributions.

## Quick Start

**Ready to contribute? Here's how:**

1. **Browse [open issues](https://github.jpl.nasa.gov/GenAI/genai-marketplace/issues)** or [start a discussion](https://github.jpl.nasa.gov/GenAI/genai-marketplace/discussions)
2. **Fork the [repository](https://github.jpl.nasa.gov/GenAI/genai-marketplace)** and make your changes
3. **Submit a pull request** with a clear description

That's it! We'll review and work with you to get it merged.

## Pull Request Guidelines

**Keep it simple:**
- **One change per PR** - Easier to review and merge
- **Reference an issue** - Link to related discussions
- **Write clear commits** - Use format: `Issue #123 - Brief description`
- **Test your changes** - Ensure everything works

**Review process:**
We'll check for clarity, correctness, and compatibility. Address feedback promptly and we'll get your contribution merged!

**First time?** Check out [this guide](https://kcd.im/pull-request) on creating pull requests.
